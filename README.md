# Backplane Scala API #

This API currently supports Backplane v1 Provisioning. It is presented
as a Scala API and SBT console helpers

Requires: SBT 0.12.2

## SBT Console ##

    git clone https://github.com/kender-janrain/bp-scala-api.git
    sbt console
    import BP._

see BP.scala for console interface
